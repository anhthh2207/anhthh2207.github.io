Research

I'm interested in computer vision, deep learning, generative AI, and image processing. Most of my research is about inferring the physical world (shape, motion, color, light, etc) from images, usually with radiance fields. Some papers are highlighted.

Pipelined RISCV Processor
project page / code

A custom 32-bit RISC-V core in SystemVerilog with a five-stage fully pipelined datapath, including full forwarding and hazard control. It features I$ and D$ that interface with main memory via the AXIL protocol, along with low-latency operators such as a Carry-Lookahead Adder and an 8-cycle divider.

Efficient DNN via Pruning & Sparse Matrix Compression
project page / code

We investigated various patterns and intensities of pruning; developed custom layer leveraging CSR format for efficient storage and inference of deep neural network.

Configurable Logic Block Design, Verification and Optimization
project page

This project focuses on optimizing Configurable Logic Blocks (CLB) design by addressing timing hazards, and power-performance trade-offs to minimize the Figure of Merit.

Auto-driving Engines for F1/10
project page / code

We designed autonomous driving systems for F1/10 racing cars, featuring SLAM, motion planning, optimal control, deep learning-based perception, and integration across simulation and hardware platforms.

Automatic Solar Tracker
code

Automatic Solar Tracker is a double-axis system that optimizes solar panel positioning using LDR sensors and a servo motor, developed on Tiva TM4C123GH6PM microcontroller.
