projects

Take a look at some of my past projects! I mainly work on computer architecture, digital design, and efficient machine learning. In my free time, I also enjoy exploring topics like computer vision, control, and robotics.

Pipelined RISC-V Processor
project page / code

A custom 32-bit RISC-V core, written in SystemVerilog, with a five-stage fully pipelined datapath, including full forwarding and hazard control. It features I$ and D$ that interface with main memory via AXIL, plus low-latency operators like a Carry-Lookahead Adder and an 8-cycle divider.

Efficient DNN via Pruning & Sparse Matrix Compression
project page / code

This project investigated different pruning patterns and sparsity levels, and developed a custom linear layer leveraging the Compressed Sparse Row (CSR) format to improve memory efficiency and accelerate neural network inference.

Configurable Logic Block Design, Verification and Optimization
project page

This project presents the design and verification of a 16-bit Configurable Logic Block (CLB) implemented using Cadence 45nm Salicide 1.0V/1.8V 1P 11M technology. Our design integrates key modules, including a 16:1 LUT, SRAM, SIPO register, and clock generator, carefully tuned to resolve timing hazards and balance power-performance.

Auto-driving Engines for F1/10
project page / code

We designed autonomous driving systems for F1/10 racing cars, featuring SLAM, motion planning, optimal control, deep learning-based perception, and integration across simulation and hardware platforms.

LC3 Simulator
code

This project implemented the LC-3 (Little Computer 3) architecture in C. We verified and demonstrated our design using games written in LC-3 Assembly, including 2048, Hangman, and Rogue.

Automatic Solar Tracker
code

Automatic Solar Tracker is a dual-axis system designed to maximize solar energy absorption by adjusting panel orientation based on input from Light Dependent Resistor (LDR) sensors. The system is controlled by servo motors powered by the Tiva TM4C123GH6PM microcontroller.
